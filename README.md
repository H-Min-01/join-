# 멀티플레이 퍼즐 웹 게임

PRD를 바탕으로 구현한 의존성 없는 MVP입니다. 방 생성, 링크/룸코드 입장, 게스트 세션, 실시간 멀티플레이, Wordle/숫자야구 로직, 상태 저장과 재접속 복구를 포함합니다.

## 실행

```bash
npm start
```

기본 포트는 `3000`입니다. 브라우저에서 `http://localhost:3000`으로 접속하면 됩니다.

## 포함 기능

- 비회원 게스트 세션 생성
- 방 생성과 룸코드 공유
- Wordle / 숫자야구 선택
- 대기실, 방장 시작, 결과 확인, 다시 하기
- WebSocket 기반 실시간 진행도 동기화
- `data/state.json` 파일 기반 상태 영속성
- 모바일 우선 반응형 UI

## 참고

- `카카오톡으로 공유` 버튼은 네이티브 공유 시트를 우선 사용합니다. 모바일에서 카카오톡이 설치되어 있으면 공유 시트에서 선택할 수 있습니다.
- Wordle은 영문 알파벳 입력 기준으로 동작합니다.
